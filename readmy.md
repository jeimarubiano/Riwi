the program runs with python, everything is within register_students.py
it has a menu with 6 options which are:
1: register a student.
it is used to register a student in which the console asks you to:
name, age, course, and whether the student is active or not
2: consult students list 
consult the list of registered students, legibly shows all registered students
3: Search a student. 
in this option it gives you a search menu that you want either by ID, name, age, course, or status, if it does not find it says that it does not exist.
4: Delete a student
delete a student using ID
5: update a student
this option is to update the information of a registered student.
6: Exit
it leaves the program
